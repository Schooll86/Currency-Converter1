# Currency-Converter1

## Автор
Обучающийся

## Описание
Приложение для конвертации валют с использованием актуальных курсов.

## Функции
- Конвертация валют (USD, EUR, RUB и др.)
- Ввод суммы пользователем
- Получение актуального курса
- Простое графическое/консольное приложение

## Требования
- Python 3.x
- requests (если используется API)

## Запуск
1. Установить Python
2. Установить зависимости:

   pip install requests

3. Запустить:

   python main.py

## Примечание
Курсы валют могут обновляться через API.
